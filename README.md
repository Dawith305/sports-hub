# Sports Hub

## Run the application

Add these environment variables (e.g. copy `.env.example` to `.env`) before starting the app:

- `VITE_THESPORTSDB_API_BASE` — TheSportsDB API base URL
- `VITE_DEFAULT_LEAGUE_ID` — League id for upcoming fixtures

Install dependencies:

```bash
pnpm install
```

Start the development server:

```bash
pnpm dev
```
